JSON Veri İşlemleri Örnekleri

Örnek 1: JSON.parse - JSON String'i JavaScript nesnesine Çevir

JSON formatındaki stringi JavaScript nesnesine dönüştürme.

const jsonString = '{"id": 1, "name": "Sağlık Eczanesi", "city": "İstanbul"}'; const obj = JSON.parse(jsonString); console.log("Nesne:", obj); console.log("Eczane adı:", obj.name);

Örnek 2: JSON.stringify - JavaScript Nesnesini JSON String'ine Çevirme

JavaScript nesnesini JSON formatına dönüştürüyoruz.

const pharmacy = { id: 2, name: "Başkent Eczanesi", city: "Ankara", isOpen: true }; const jsonString = JSON.stringify(pharmacy); console.log("JSON String:", jsonString);

Örnek 3: Nesne Üzerinde Değişiklik Yapma

JSON'dan nesneye çevirdikten sonra nesnenin özelliklerini değiştiriyoruz.

const jsonData = '{"id": 3, "name": "Deniz Eczanesi", "phone": "0232-555-9012"}'; const pharmacy = JSON.parse(jsonData); console.log("Orijinal:", pharmacy); // Değişiklik yap pharmacy.name = "Deniz Eczanesi (Güncellenmiş)"; pharmacy.phone = "0232-555-0000"; console.log("Değiştirildikten sonra:", pharmacy); // Tekrar JSON'a dönüştür const updatedJson = JSON.stringify(pharmacy); console.log("Güncellenmiş JSON:", updatedJson);

Örnek 4: Dizi İçinde Nesne Değişikliği

İlaç dizisinde bir ilaç stokunu güncellemeliyiz.

const jsonData = '[{"id": 101, "name": "Paracetamol", "stock": 120}, {"id": 102, "name": "Amoksisilin", "stock": 85}]'; const medicines = JSON.parse(jsonData); console.log("Orijinal İlaçlar:", medicines); // Stok güncelle medicines[0].stock = 95; medicines[1].stock = 50; console.log("Güncellenmiş İlaçlar:", medicines); // JSON'a dönüştür const updatedJson = JSON.stringify(medicines, null, 2); console.log("Güncellenmiş JSON:\n" + updatedJson);

Örnek 5: Diziye Yeni Nesne Ekleme ve JSON'a Dönüştürme

Eczane listesine yeni bir eczane ekliyoruz.

const jsonData = '[{"id": 1, "name": "Sağlık Eczanesi", "city": "İstanbul"}, {"id": 2, "name": "Başkent Eczanesi", "city": "Ankara"}]'; const pharmacies = JSON.parse(jsonData); console.log("Orijinal Eczaneler:", pharmacies); // Yeni eczane ekle const newPharmacy = { id: 4, name: "Marmara Eczanesi", city: "Bursa" }; pharmacies.push(newPharmacy); console.log("Yeni eczane eklendikten sonra:", pharmacies); // JSON'a dönüştür const finalJson = JSON.stringify(pharmacies, null, 2); console.log("Final JSON:\n" + finalJson);

Örnek 6: Karmaşık Nesne İşlemi (İç İçe Veri)

Eczane ve ilaçlar gibi iç içe geçmiş verileri işleyerek güncellemeler yapıyoruz.

const complexData = '{"pharmacy": {"id": 1, "name": "Sağlık Eczanesi"}, "medicines": [{"id": 101, "name": "Paracetamol", "price": 15.50}, {"id": 102, "name": "Ibuprofen", "price": 22.00}]}'; const data = JSON.parse(complexData); console.log("Orijinal Veri:", data); // Değişiklikler yap data.pharmacy.name = "Sağlık Eczanesi - Şubesi"; data.medicines[0].price = 14.50; data.medicines[1].price = 20.00; console.log("Güncellenmiş Veri:", data); // JSON'a dönüştür const updatedJson = JSON.stringify(data, null, 2); console.log("Final JSON:\n" + updatedJson);
