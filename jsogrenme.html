Event Listener öğrenme sayfası

addEventListener ile click, submit ve diğer olayları dinle ve işle

1. Temel Click Event Dinleme

Amaç: Bir butona tıklandığında fonksiyon çalıştırma.

Kullanılan metodlar:

- addEventListener("click", ...) - Tıklama olayını dinle

- function(event) {} - Olayda tetiklenen fonksiyon

- console.log() - Konsola veri yazdırma

Sonuç:
const btn = document.getElementById("clickBtn"); btn.addEventListener("click", function(event) { console.log("Butona tıklandı!"); console.log("Event türü:", event.type); console.log("Tıklanan element:", event.target); });

2. Event Nesnesi (event.target, event.type)

Amaç: Event nesnesi özelliklerini kullanarak olayı kontrol etme ve bilgi alma.

Event özellikleri:

- event.type - Olayın türü (click, submit, vb.)

- event.target - Olayı tetikleyen element

- event.timeStamp - Olayın meydana geldiği saat

Event Bilgisi:
// Tüm butonlara event listener ekle const buttons = document.querySelectorAll("[id^='eventBtn']"); buttons.forEach(function(btn) { btn.addEventListener("click", function(event) { console.log("Olayın türü:", event.type); console.log("Tıklanan buton:", event.target.id); console.log("Buton metni:", event.target.innerText); }); });

3. preventDefault() - Varsayılan Davranışı Engelleme

Amaç: Event nesnesinin varsayılan davranışını engelleme.

Örnekler:

- Linke tıklandığında sayfanın açılmasını engelleme

- Formun varsayılan şekilde gönderilmesini engelleme

- Checkbox'ın işaretlenmesini engelleme

Bu Linke Tıkla (Sayfa Açılmayacak)
Sonuç:
const link = document.getElementById("exampleLink"); link.addEventListener("click", function(event) { event.preventDefault(); // Varsayılan davranışı engelle console.log("Link tıklandı ama sayfa açılmadı!"); console.log("Hedef URL:", event.target.href); });

4. Submit Event - Form Gönderme Olayını Dinleme

Amaç: Form gönderilirken (submit) olayı yakala ve verileri kontrol et.

Kullanılan metodlar:

- addEventListener("submit", ...) - Form submit olayını dinle

- event.preventDefault() - Sayfanın yenilenmesini engelle

- form.elements - Form elemanlarına erişme

Form Verileri:
const form = document.getElementById("submitForm"); form.addEventListener("submit", function(event) { event.preventDefault(); // Sayfanın yenilenmesini engelle // Form verilerini al const name = document.getElementById("submitName").value; const age = document.getElementById("submitAge").value; console.log("Form gönderildi!"); console.log("Adı:", name); console.log("Yaşı:", age); });

5. Input Event - Gerçek Zamanlı Veri İzleme

Amaç: Input alanına yazıldıkça değişiklikleri gerçek zamanlı izleme.

Event türleri:

- input - Değer her değiştiğinde tetiklenir

- change - Element odaktan çıktığında tetiklenir

Yazılan Metin:
Henüz birşey yazılmadı
const input = document.getElementById("liveInput"); input.addEventListener("input", function(event) { const value = event.target.value; console.log("Yazılan metin:", value); console.log("Metin uzunluğu:", value.length); });

6. Event Bubbling ve stopPropagation()

Amaç: Event bubbling (olayların yukarı yayılması) olayını anlamak ve kontrol etmek.

Event Bubbling: Bir elemanın olayı tetiklenmesi halinde, aynı olay parent elemanlarına da yayılır.

- event.stopPropagation() - Olayın yayılmasını durdur

Üst Div (Kırmızı)

Event Bilgisi:
const parentDiv = document.getElementById("parentDiv"); const childBtn = document.getElementById("childBtn"); // Ebeveyn divine event listener ekle parentDiv.addEventListener("click", function(event) { console.log("Parent div tıklandı!"); }); // Çocuk butona event listener ekle childBtn.addEventListener("click", function(event) { event.stopPropagation(); // Olayın yayılmasını durdur console.log("Buton tıklandı (üst olaya yayılmadı!)"); });
Not: stopPropagation() olmadan, buton tıklanırsa hem buton hem de parent div'in olayları tetiklenir.
